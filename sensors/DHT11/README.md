# DHT11
![image-removebg-preview (3)](https://github.com/cyberdebb/estacao_meteorologica/assets/107296659/1c9f498e-6577-4998-99b0-00524eba8f21)

The [DHT11](https://www.makerhero.com/img/files/download/DHT11-Datasheet.pdf) is a digital-output temperature and humidity sensor that provides high reliability and long-term stability. The temperature sensing element is an NTC-type thermistor, and the humidity sensor is an HR202 type. The internal circuit reads the sensors and communicates with a microcontroller through a single-wire serial signal.

## Specifications
| INFORMATION                          | DHT11                         |
| ----------------------------------- | ----------------------------- |
| POWER SUPPLY                         | 3.0 to 5.0 VDC (5.5 VDC max)  |
| HUMIDITY MEASUREMENT RANGE           | 20 to 95% RH                  |
| TEMPERATURE MEASUREMENT RANGE        | 0° to 50°C                    |
| HUMIDITY MEASUREMENT ACCURACY        | ±5.0% RH                     |
| TEMPERATURE MEASUREMENT ACCURACY     | ±2.0 °C                      |

## Connections
| DHT | ESP WROOM 32 | ESP32 Pin Number |
|:---:|:------------:|:---------------:|
| VCC | 3V3          | 3V3             |
| GND | GND          | GND             |
| DAT | GPIO         | GPIO15          |
